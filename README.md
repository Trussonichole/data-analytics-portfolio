## Projects

### Retail Sales Data Analysis
**SQL Query:** Retail Sales Analysis  
**Tools:** SQL, BigQuery  

Analyzed retail transaction data to uncover actionable insights in revenue and customer behavior.

**Key analysis included:**
- **Revenue trends:** Calculated monthly and country-level revenue to identify top-performing markets.
- **Customer behavior:** Measured purchase frequency to understand repeat customers.
- **Product performance:** Assessed which products contributed most to sales.

**Dataset:** Analysis performed on the full [Online Retail Dataset from Kaggle](https://www.kaggle.com/datasets/competitions/online-retail-data) using **Google BigQuery**. A small sample is provided in the repository for reference, while all queries ran on the full dataset for accurate insights.

**Location in repo:** ``/sql/revenue_by_country.sql``
